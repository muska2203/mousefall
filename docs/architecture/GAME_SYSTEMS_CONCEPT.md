## Статус концепта и соответствие коду

Этот файл — **черновик / концепт**. Не все описанные здесь эффекты и комбо реализованы.

### Уже реализованные статусы

Реализованные в коде (`StatusEffectType` в `src/simulation/core-types.ts`):

- `poisoned` — Отравление
- `burning` — Горение
- `frozen` — Заморозка
- `stunned` — Оглушение
- `silenced` — Безмолвие
- `regenerating` — Регенерация
- `counterattack` — Эффект контратаки

Остальные эффекты (`panic`, `bleeding`, `barrier`, `rage`, `wet`, `oil`, `sleep` и др.) пока не реализованы и описаны как план.

### Классификация урона: от `DamageType` к тегам

Согласованный переход типа урона из enum/union в **иерархические игровые теги**:

- Физический урон: `damage.physical.piercing`, `damage.physical.slashing`, `damage.physical.blunt`
- Магический урон: `damage.magical.fire`, `damage.magical.electric`, `damage.magical.poison`, `damage.magical.frost`

Теги проверяются через `hasTag` / `hasAllTags` / `hasAnyTag` (`src/simulation/systems/tags/tag-helpers.ts`). Родительские теги выводятся автоматически: `damage.physical.slashing` удовлетворяет `damage.physical`.

`DamageType` в `src/simulation/core-types.ts` и поле `damageType` в JSON-шаблонах сохраняются для совместимости с интентами и событиями, но новые механики должны опираться на теги.

### Броня

Броня применяется только к физическому урону (тег `damage.physical`). Магический урон игнорирует броню, если в обработчике не указано иное.

## Система активных эффектов (Status Effects)

Активные эффекты — временные или постоянные состояния, дающие актёру преимущества и недостатки.

### Общие правила системы эффектов

- **Один экземпляр на цель**: на одном акторе может висеть только один экземпляр конкретного эффекта.
- **Замена при повторном наложении**: новый эффект заменяет старый, если он сильнее или длительнее (правило сравнения уточняется ниже).
- **Глобальный тик**: эффекты, наносящие урон или изменяющие состояние, срабатывают в фиксированный момент игрового цикла (не привязан к ходу конкретного актора).

### Негативные эффекты

| Эффект | Краткое описание |
|--------|-----------------|
| **Горение** | Наносит урон, с небольшим шансом вызывает панику |
| **Отравление** | Наносит урон меньше горения. Каждый шаг при перемещении добавляет +1 ход к длительности |
| **Паника** | Вместо своего хода делает случайные передвижения, тратя все AP. Может уклониться от входящего урона |
| **Оглушение** | Полностью пропускает ход (AP=0). Получает увеличенный урон |
| **Кровотечение** | Негативный статус. Тикает 3 хода, урон за тик = 50% от удара, вызвавшего кровотечение |
| **Раскол брони** | Снижает броню фиксированным значением, стакается (до максимума). Не опускает ниже 0 |

### Позитивные эффекты

| Эффект | Краткое описание |
|--------|-----------------|
| **Барьер** | Имеет собственные HP. Поглощает магический урон, дополнительно снижая его. Разрушается физической атакой. При разрушении взрывается, нанося урон носителю и всем в радиусе 1 клетки |
| **Ярость** | Наносимый урон +50%, получаемый урон +50% |
| **Перегрузка** | +1 AP на 2 хода. После — 2 хода с заблокированным движением (AP остаются, можно атаковать/использовать предметы) |
| **Вампиризм** | При ≤30% HP: 50% нанесённого урона возвращается как лечение. Все остальные источники лечения эффективны на 50% |
| **Укрепление** | +2 брони в начале каждого хода (макс +6). Движение заблокировано |

### Нейтральные эффекты

| Эффект | Краткое описание |
|--------|-----------------|
| **Мокрый** | Длится 3 хода. Любой огненный урон снимает эффект и не вызывает горения. Тушит существующее горение |
| **Масло** | +20% уклонение. При поджоге — большой разовый урон только по носителю |
| **Отложенный урон** | Весь входящий урон игнорирует броню. 1/3 наносится сразу, оставшиеся 2/3 — равными долями на следующие 2 хода |
| **Сон** | Негативный статус. Пропускает ход (AP=0), восстанавливает HP каждый ход. Снимается любым входящим уроном |
| **Сонливость** | Нейтральный статус. Через 3 хода переходит в **Сон**. Снимается входящим уроном до перехода |
| **В тумане** | +30% уклонение, -30% к точности атак по цели в тумане |

## Комбо-взаимодействия: статусы × типы урона

При определённых статусах на цели тот или иной тип урона даёт дополнительный эффект.

### Огненный урон

| Статус цели | Результат |
|-------------|-----------|
| **Мокрый** | Тушит мокрый. Урон снижен на 50%, горение не накладывается |
| **Масло** | Взрыв: большой разовый урон только по цели. Снимает масло |
| **Горение** | Усиливает горение: +1 ход длительности, +50% урона следующего тика |
| **Ярость** | Разгорается сильнее: +25% к урону огнём по цели в ярости, ярость не снимается |
| **В тумане** | Рассеивает туман в радиусе 2 клеток от цели |
| **Под морозным эффектом** | Пар: оба эффекта снимаются, цель получает **Ослепление** на 1 ход (−30% точности) |

### Электрический урон

| Статус цели | Результат |
|-------------|-----------|
| **Мокрый** | Цепная молния: урон +50%, поражает ещё 1 случайную цель в радиусе 2 клеток |
| **В тумане** | Распространяется по туману: урон делится между всеми целями в тумане в радиусе 3 клеток |
| **Барьер** | Барьер поглощает, но при разрушении взрыв наносит электрический урон вместо физического |
| **Оглушение** | Шок: шанс 50% продлить оглушение на 1 ход |
| **Перегрузка** | +1 ход к длительности перегрузки (и бонус AP, и штраф длиннее) |
| **Масло** | Искра поджигает масло: масло превращается в **Горение** |

### Ядовитый урон

| Статус цели | Результат |
|-------------|-----------|
| **Отравление** | Усиливает яд: урон отравления +50%, длительность +1 ход |
| **Кровотечение** | Инфекция: кровотечение превращается в **Отравление** (урон сохраняется, механика — яд) |
| **Мокрый** | Разбавление: ядовитый урон снижен на 30%, но мокрый снимается |
| **Ярость** | Бешенство: ярость усиливается (+10% к урону), но после окончания Ярости цель получает **Панику** |
| **Барьер** | Барьер блокирует яд, но яд оседает: при разрушении барьера взрыв отравляет всех в радиусе |
| **В тумане** | Газ: ядовитый урон по цели в тумане накладывает отравление на всех в тумане |

### Морозный урон

| Статус цели | Результат |
|-------------|-----------|
| **Мокрый** | Заморозка: мокрый превращается в **Обморожение** (не может двигаться 1 ход, броня +2) |
| **Горение** | Пар: оба эффекта снимаются, создаёт туман в радиусе 2 клеток на 2 хода |
| **Масло** | Застывание: масло превращается в липкое масло (+30% уклонение, но поджог = более сильный взрыв) |
| **Оглушение** | Криостазис: оглушение +1 ход, но цель получает на 50% меньше урона |
| **В тумане** | Иней: туман замерзает, все в тумане получают **Обморожение** |
| **Отравление** | Консервация: отравление замирает (не тикает), но при огненном уроне — взрыв яда |

### Колющий урон

| Статус цели | Результат |
|-------------|-----------|
| **Кровотечение** | Углубление раны: урон за тик кровотечения +50% |
| **Барьер** | Пробой: игнорирует броню и пробивает сквозь барьер (урон по цели, барьер не поглощает) |
| **Укрепление** | Игнорирует броню от укрепления, но НЕ снимает его |
| **Сон** | Крит по спящему: +100% урона, но Сон снимается |

### Рубящий урон

| Статус цели | Результат |
|-------------|-----------|
| **Кровотечение** | Рассечение: длительность кровотечения +1 ход |
| **Масло** | Скользкий удар: шанс промаха +20%, но при попадании кровотечение гарантировано |
| **Укрепление** | Трещина: снимает 1 стак укрепления (−2 брони) |
| **Барьер** | Разрушение: разрушает барьер, но урон по цели −30% |

### Тупой урон

| Статус цели | Результат |
|-------------|-----------|
| **Оглушение** | Добивание: урон по оглушённой цели +50% |
| **Укрепление** | Сбивание с толку: шанс 50% снять укрепление полностью |
| **Барьер** | Гарантированно разрушает барьер |
| **Сон** | Пробуждение без бонуса: обычный урон, Сон снимается |
