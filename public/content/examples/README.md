# Примеры шаблонов контента

> Эти файлы — образцы для копирования. Они не добавлены в `manifest.json` и не загружаются игрой.

Использование:

1. Скопируйте нужный шаблон в соответствующую папку `public/content/`.
2. Замените `id` и остальные значения.
3. Добавьте путь в `public/content/manifest.json`.
4. Добавьте тексты в `src/content/texts/{ru,en}/`.

Шаблоны:

- [`enemy-template.json`](./enemy-template.json) — враг.
- [`weapon-template.json`](./weapon-template.json) — оружие.
- [`status-template.json`](./status-template.json) — статус.
- [`ability-template.json`](./ability-template.json) — способность.
